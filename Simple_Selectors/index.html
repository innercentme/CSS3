Introduction to CSS

This is an example HTML document to demonstrate simple CSS selectors.

Selectors

In CSS, selectors are used to target specific elements for styling.

The Universal Selector (*)

The universal selector targets all elements on the page.

Element Selector (e.g., p)

The element selector targets specific HTML elements. For example, p selects all paragraph elements.

Class Selector (e.g., .classname)

The class selector targets elements with a specific class attribute value. For example, .highlight selects all elements with class="highlight".

ID Selector (e.g., #idname)

The ID selector targets an element with a specific ID attribute value. For example, #header selects the element with id="header".

Descendant Selector (e.g., p span)

The descendant selector targets elements that are descendants of another element. For example, p span selects all span elements that are descendants of p elements.
