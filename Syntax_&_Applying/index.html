Introduction

This course is designed for beginners who want to learn the basics of CSS.

Basic CSS Syntax

In CSS, we use selectors to target elements and rules to apply styles.

For example, to change the text color to red, we can use the following rule:

            selector {
                color: red;
            }

Applying Styles in CSS

Styles can be applied directly to HTML elements using the style attribute.

            <p style="color: blue;">This text will be blue.</p>

However, it's better to use external CSS files that we link to the HTML document using the <link> tag.

For example, we have a file named "styles.css" where we define styles for our document.

Add this tag to your HTML document to include the styles:

            <link rel="stylesheet" href="styles.css">
